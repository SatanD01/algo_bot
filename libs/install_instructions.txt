# Инструкция по установке проекта

## 1. Установка Python
Перед началом работы убедитесь, что у вас установлен Python 3.9 или выше.
Скачать можно с официального сайта: https://www.python.org/downloads/

Во время установки **обязательно** установите галочку **"Add Python to PATH"**.

## 2. Установка необходимых зависимостей
Откройте терминал или командную строку в папке с проектом и выполните команду:

```
pip install -r requirements.txt
```

Если у вас Windows и возникли проблемы с установкой MetaTrader5, попробуйте:

```
pip install --upgrade pip setuptools wheel
pip install MetaTrader5 --no-cache-dir
```

## 3. Настройка конфигурации
Откройте файл `config.py` и укажите:

- Логин и пароль от вашего торгового счета MT5.
- Сервер брокера.
- Торгуемый инструмент (по умолчанию EUR/USD).
- Параметры стратегии, включая риск, таймфреймы и торговые ограничения.

## 4. Запуск бэктеста
Чтобы запустить бэктест:

```
python main.py
```

Если хотите запустить оптимизированный бэктест:

```
python main.py --optimized
```

## 5. Запуск реального трейдинга
Чтобы запустить бота в реальном времени, убедитесь, что **MODE** в `config.py` установлен в `"live"`:

```python
MODE = "live"
```

Затем запустите команду:

```
python main.py --live
```

## 6. Просмотр логов
Логи работы бота сохраняются в `logs/`. Чтобы посмотреть последние записи, выполните:

```
tail -f logs/bot_logs.txt  # Для Linux/macOS
type logsot_logs.txt     # Для Windows
```

## 7. Визуализация результатов
После завершения бэктеста можно построить графики результатов:

```
python visualization.py
```

Это создаст файлы графиков в папке `backtest_results/`.
